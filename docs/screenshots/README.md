# Screenshots

Drop your screenshots here with these **exact filenames** — the main `README.md`
embeds them by these paths.

| Filename | Screen | Source |
|----------|--------|--------|
| `web-dashboard.png` | Web 儀表板 / 飲食頁（熱量卡、餐點列表） | 桌面瀏覽器截圖 |
| `app-dashboard.png` | App 儀表板（當日攝取、淨熱量） | 手機截圖 |
| `app-daily-summary.png` | App「昨日總結」自動彈窗 | 手機截圖 |
| `app-health-sync.png` | App 健康同步 / 設定頁 | 手機截圖 |

Tips:
- Web: 視窗寬度約 1200–1440px，截主要內容區即可。
- App: 直接用手機截圖（約 1080×2400），README 會自動縮放並排顯示。
- 建議用 PNG；檔案過大可先壓縮（例如 TinyPNG）。
- 換圖時保持相同檔名即可，不需改動 `README.md`。
